CSS Pseudo-elements..!

The css pseudo-elements allow you to style the elements or parts of element without adding any ID's or classess to them

CSS3 introduced a new double icon(::) syntax for pseudo-elemets to distinguish between then and pseudo-classess.

selector::pseudo-elemets {property:value};
